## HOW TO USE
The macOS build agent MUST:
1. Before implementing any screen: `grep "Screen: <name>" FEATURE_PARITY_REGISTRY.md` to surface every item
2. Implement every ❌ item — no exceptions
3. Change ❌ → ✅ (implemented) or 🔄 (adapted for desktop, with note in `macOS Impl` column)
4. Do NOT mark a screen complete until zero ❌ items remain for it
5. Output `✅ <ScreenName> complete — N implemented, M adapted, 0 skipped` after each screen
