# LocalOCR macOS

Native SwiftUI + AppKit client for [LocalOCR Extended](../). Target: macOS 13 Ventura+, Apple Silicon.

This subtree is built per [`../MACOS_APP_PLAN.md`](../MACOS_APP_PLAN.md) and [`../VETO_RESOLUTION_PATCH.md`](../VETO_RESOLUTION_PATCH.md).

---

## Phase 1 status — Foundation scaffolded, awaiting Xcode

Phase 1 (§5.8) is complete except for the final verify step. The `.xcodeproj` is **not** committed — it's generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `project.yml` so the project structure stays declarative and reviewable.

### To resume (one-time setup)

```sh
# 1. Install Xcode 15.0+ from the Mac App Store (~12GB). Then:
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
xcode-select -p  # should print /Applications/Xcode.app/Contents/Developer

# 2. Install XcodeGen:
brew install xcodegen

# 3. Generate the Xcode project from project.yml:
cd LocalOCR.macOS
xcodegen generate

# 4. Open in Xcode:
open LocalOCR.xcodeproj
```

### Phase 1 verify step (run after generating):

```sh
cd LocalOCR.macOS
xcodebuild -scheme LocalOCR -configuration Debug build
open ./build/Debug/LocalOCR.app  # or hit ⌘R in Xcode
```

**Expected:** Empty window opens titled "LocalOCR" showing "Phase 1 — Foundation".

## Local-only build

Per build-agent direction: **unsigned local builds.** `project.yml` sets:

```
CODE_SIGN_IDENTITY: "-"
CODE_SIGNING_REQUIRED: NO
CODE_SIGNING_ALLOWED: NO
```

Code signing + notarization + DMG distribution are deferred to Phase 9 and require an Apple Developer Program account.
