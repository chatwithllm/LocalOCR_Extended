Contribution

See how scoring works, what you earned, and how you can help the system stay sharp.

How Low-Stock Validation Works
1. Mark an item Low. This starts a pending contribution, but it does not score yet.
2. Move that item into the Shopping List. That helps the system connect the restock intent.
3. When a later receipt confirms the item was bought, the low-stock action is validated and earns better points.
4. If the low flag is quickly cleared or turns out to be wrong, the pending contribution does not count.
